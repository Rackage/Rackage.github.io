RACKAGE

Roblox Package Manager



Rackage is a brand new package manager for Roblox. It allows you to install uploaded packages inside studio using our plugin without the need for an external program (although we do plan on this in the future).

You can also install packages directly from our website and you'll be prompted to add them to whichever game you are currently editing in studio.

Rackage is entirely open-source, so you can edit it to your liking!
